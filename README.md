## 1.疫情数据抓取

通过爬虫获取疫情的数据，最早可获取到1月份至今的数据。

数据包括：
  1. 当日数据
  2. 历史数据(时间序列)
  
数据地址：https://wp.m.163.com/163/page/news/virus_report/index.html

代码地址：[基于网易疫情播报平台的数据采集](https://github.com/SalientView/covid_19-data-crawler)
